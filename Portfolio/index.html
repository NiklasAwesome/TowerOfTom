TDP005 Projekt: Objektorienterat system

Detta spel heter Tower of Tom.
Tower of Tom är ett tvådimentionellt arkadspel där målet är att ta sig igenom alla labyrinter på så kort tid som möjligt utan att dö. Spelet spelas med hjälp av piltangeterna eller W/A/S/D och mellanslag för att attackera. Det kommer finnas väggar på vissa rutor som man inte kan röra sig igenom. Det kommer vara massa fiender ivägen för spelaren som spelaren ska ta sig förbi eller förinta.

Använda tekniker:

  • C++
  • SFML
  • Make
  • Git
